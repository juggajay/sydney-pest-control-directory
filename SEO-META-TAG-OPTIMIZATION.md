## OPTIMIZATION RECOMMENDATIONS

### Character Count Best Practices
- **Title**: 50-58 chars (Google's sweet spot for full display)
- **Description**: 155-160 chars (Maximum useful length)
- **Primary keyword**: First 30 chars of title for mobile
- **Call-to-action**: Must appear before character 130 in description

### Power Words to Use
- **Urgency**: Fast, Same-day, Immediate, Emergency, Urgent
- **Social Proof**: Trusted, 4.8â˜…, Top-rated, Verified, Licensed
- **Emotional**: Safe, Protected, Reliable, Professional, Guaranteed
- **Action**: Get, Find, Compare, Book, Request, Call Now

### Local SEO Priorities
1. Suburb name in title (within first 30 chars on mobile)
2. Service area coverage number as trust signal
3. Rating/review count visible in snippet
4. CTA before truncation point

### Character Validation Function
```javascript
export function validateMetaTags(title, description) {
  const warnings = [];

  if (title.length < 30) warnings.push("Title too short (<30 chars)");
  if (title.length > 60) warnings.push(`Title too long (${title.length} chars, will truncate)`);

  if (description.length < 120) warnings.push("Description too short (<120 chars)");
  if (description.length > 160) warnings.push(`Description too long (${description.length} chars)`);

  if (title.length > 35 && !title.substring(0, 35).includes("Sydney")) {
    warnings.push("Location not in first 35 chars (mobile risk)");
  }

  const cta = description.match(/\b(get|call|request|find|book|contact)\b/i);
  if (!cta) warnings.push("No clear CTA verb found");

  const ctaPosition = description.lastIndexOf(cta?.[0] || '');
  if (ctaPosition > 130) warnings.push("CTA after char 130 (mobile truncation risk)");

  return { isValid: warnings.length === 0, warnings };
}
```
